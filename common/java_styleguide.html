Java Style Guide

Background

Activity

As a group, download and review the Java stype guide from Oracle.

Using the sample checklist in the discussion as a starting point, create a checklist for your team to use when reviewing code. This checklist should be followed for your own code and will be included in the rubric for your coding exercises.

  • Code is compliant with accepted style guide.
  • Identifiers, including filenames, classes, variables, constants, and methods are descriptive, meaningful, and follow naming conventions.
  • Code uses correct line breaks and whitespace
  • Projects, Files, Classes, and methods use correct javadoc syntax.
