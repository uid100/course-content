Troubleshooting

Importing modules: Why doesn't my import ##### statement work?

Recently installed

Most external modules will be installed using PIP. Before you can use PIP, you need to know where Python is installed on your system.

Start with a terminal window.

From Windows, you can use the Command Prompt or PowerShell. 

                            C:\ > python --version
                            Python 3.13.3

From macOS, you can use the Terminal app. 

                            % python3 --version
                            Python 3.13.3

As of today, the latest version of Python (shown) indicates the third maintenance release of Python 3.13. You may have a different version installed. If it's recent, it should be ok.

Not what I expected

I installed Python, but the command doesn't work.

There are a few reasons why the command might not work.

If you just installed Python, the PATH environment variable may not be updated, or it could not be recognized by the terminal. The command shell, either in Windows or macOS, needs to know where to find the Python executable. The shell itself only reads this environment variable when it starts up. So the first thing to try is to close the terminal window, or command prompt, and open a new one. Then try the command again.

If it still isn't working, make sure that the Python executable is in the PATH environment variable. You can check this by running the following command: 

                            C:\ > echo %PATH%
                            C:\Python39\;C:\Python39\Scripts\;...

The equivalent on macos is

                            % echo $PATH
                            /usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin:...

If you see the path to the Python installation, then the command should work. If not, you may need to add it manually.

Also, on some systems, python may be linked to python3, or python3.13, or sometimes py.

If you have multiple versions of Python installed, including Python 2.x, you may need to use python3 or py to run the correct version.

If you have more than one version installed and just want to check the version of Python 3, try which python on macOS or Linux, or where python on Windows. This will show you the path to the Python executable that is currently being used.

    Edit the source code

    Replace the source code with the following and update the header with your information.:

    The program should randomly generate a number of apples for each of three players. The program should then compare the results using if-else statements to determine the winner.

  2. Explanation

    Understanding the code

    Random

    The program uses the Random class to generate random numbers between 1 and 100 for each of the three players. The program then compares the results using if-else statements to determine the winner.

    The code first creates a new instance of the Random class and assigns it to the variable random. We need to do this just once at the beginning of the program, so we can use it to generate random numbers.

    The Next() method is then called on the random object to generate a random number between 1 and 100.

    The Next() method takes two arguments: the minimum and maximum values for the random number. In this case, the minimum value is 1 and the maximum value is 101. The maximum value is exclusive, so the random number will be between 1 and 100.

    Boolean control statements

    This is a very simple program, but it demonstrates the use of the conditional if statement, the else if statement, and the else statement. The program also uses the Console.WriteLine()

    Remember that an else-if statement, just like an if statement requires a boolean expression. The else statement does not.

    Also, each of the boolean expressions evluates not just two integer values with a boolean operator, but also against a third integer value using the && operator. This is called a compound boolean expression.

    The program uses the familiar "Press any key..." statement to pause the program and wait for the user to press a key before exiting. This is a common practice in console applications to allow the user to see the output before the program closes.

  3. Run and test the program

    Run the program by clicking the green arrow (Start button) in the toolbar or by pressing Ctrl + F5. Or you can use either of the Start options in the Debug menu.

    Confirm that the program runs and assigns random values to the three people and that the logic correctly determines the one with the most apples.
