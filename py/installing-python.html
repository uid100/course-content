Installing Python

Background

Applications written in Python are executed by the Python interpreter. The Python interpreter is a program that reads Python code and executes it. The Python interpreter is available for all major operating systems: Windows, MacOS, and Linux.

Python is an interpreted language, which means that the Python interpreter reads and executes the code line by line. This is different from compiled languages like C++ or Java, where the code is compiled into an executable file before it is run.

Before installing Python, check to see if it is already installed on your computer. Open a command prompt or terminal window and type python --version or python3 --version. If Python is installed, you will see the version number. If Python is not installed, you will see an error message.

Note: if you are using Windows, you may be asked to install Python from the Microsoft Store. This is not recommended. Instead, download Python from the official Python website.

Download and Install Python

To install Python, follow these steps:

  1. Open a web browser and go to the Python download page.

    The Python download page will automatically detect your operating system and display the appropriate download link for the your system. There are also links to download the latest version of Python for other operating systems.

  2. Click on the Download Python button to download the latest version of Python for your computer. This installation download should not take long.
  3. Run the installer and follow the on-screen instructions to install Python on your computer. Be sure to check the box that says Add Python to PATH. The option to run as administrator is optional, but not necessary.
  4. Once the installation completes, again this should not take long. You have an option to disable the path length limit. This is optional. If you are using very long pathnames, it may be useful, but this can usually be managed by organizing your files and folders so it is generally not necessary.
  5. You can close the installer.

That's it! You now have Python installed on your Windows 11 computer.

Running IDLE

Python's Integrated Development and Learning Environment

Python comes with an Integrated Development and Learning Environment (IDLE) that provides a graphical user interface for writing and running Python code. IDLE is included with the Python installation and is available on all major operating systems.

To run IDLE, follow these steps:

    From Windows:

  1. Open the Start menu and type idle in the search box.

  2. Click on the IDLE (Python 3.x.x) icon to launch IDLE.

  3. Once IDLE is open, you can start writing and running Python code in the IDLE editor window.

    From MacOS:

  1. Open the Applications folder and double-click on the IDLE icon to launch IDLE.

  2. Once IDLE is open, you can start writing and running Python code in the IDLE editor window.

That's it! You are now ready to start writing and running Python code in IDLE.

  1. Using the Menu items, create a new file and save it as hello.py.
  2. Write the following code in the editor window: 
    print("Hello, World!")
  3. Run the code by selecting Run → Run Module from the menu.
  4. The output Hello, World! should appear in the IDLE shell window.
