Background:
Introduction to the Python Interpreter

Python is a powerful and easy-to-use programming language.

The Python interpreter allows you to execute Python commands interactively or using scripted commands stored in source files.

Let's look at different ways to invoke the Python interpreter and execute some simple commands.

Python code runs in the Python interpreter.
Using the Python Interpreter
Invoking the Interactive Interpreter

REPL is an acronym for Read, Evaluate, Print, and Loop

Python commands can be executed by entering them directly in the Python interpreter. This mode is called the Read-Evaluate-Print-Loop (REPL).

To start the Python interpreter, simply type python or python3 in your terminal, depending on your system configuration. Once started, you will see a prompt (e.g., >>>) where you can enter Python commands. 

    $ python3
    Python 3.x.x (default, yyyy-mm-dd, HH:MM:SS)
    [GCC x.x.x] on linux
    Type "help", "copyright", "credits" or "license" for more information.
    >>>

Try typing these simple examples of commands in this interactive mode: 

    >>> print("Hello, World!")
    Hello, World!
    
    >>> 2 + 2
    4
    
    >>> name = "Alice"
    >>> print(f"Hello, {name}!")
    Hello, Alice!
Using Python from the Command Line
Run a Python command from the terminal

You can also execute a Python command directly from the terminal without entering the interactive mode by using the -c option. This is useful for quick tasks. If the path is configured correctly, you can run the Python command from Windows command line or the terminal in macos or a Linux shell.

Try this simple example:

    $ python3 -c "print('Hello from the command line!')"
    Hello from the command line!

And another example performing arithmetic:

    $ python3 -c "print(5 * 10)"
    50
Running a Python (.py) file
Running a Python file from the terminal

Running a Python file from the command line cn be done by calling the Python interpreter with the file name as an argument.

Steps:

  1. Open your terminal or command prompt.
  2. Navigate to the folder where your Python file is saved.
    Use the cd command to change directories. 
        cd path/to/your/file
  3. Run the python file with the interpreter.
        python filename.py
    Replace filename.py with the name of your Python file.

Example
If you have a Python file called TaxCalculator.py with this code: 

    python TaxCalculator.py

The Python interpreter will execute the code in the file and display the output.

 

The important message is that Python runs your code, not the operating system.

Clicking on your Python file will probably not do what you expect it to.
