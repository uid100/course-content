Installing Visual Studio

This is a simple step-by-step walk through to get Visual Studio 2022 up and running on your Windows computer. If you are running another operating system, you'll may have more work to do.

 

Microsoft Visual Studio is an Integrated Development Environment (IDE).

  • Integrated 
  • Development
  • Environment

More than just an editor, or a compiler, or a code interpreter, or a debugger, or a configuration manager, or a database manager, or cloud deployment engine, or web application server. The IDE has those capabilities and more. The more you become familiar with the tools, the more useful they become. The good news is that you don't have to know about all of them to start using them and we'll walk through and gain insight as they become more familiar and comfortable.

 

Community Edition

The Community Edition is a feature-rich toolset that is free to use for students and individual developers. Other licensed environments are available to larger enterprises. If you are installing for personal use, this is it!

 

  1. Find the installer
    • Search for "Visual Studio Downloads"
    • Visual Studio 2022 is currently the latest stable release
    • Download directly from Microsoft to ensure that you are compliant with licenses
  2. Download the installer
    • Selecting Community Edition will start the download
    • Although other releases offer a "Free Trial," installing the Community Edition now will avoid unnecessary updates, license issues, and reinstalling later
  3. Launch the installer
    • There is a lot of good information on this page. Feel free to bookmark it, or browse it after you have started the installation. Once you've found this page, the download should have already started
    • Note, this has not downloaded Visual Studio, it downloaded the Visual Studio Installer
    • Once you launch the installer, you may be asked to allow the application to make changes to your computer. Answer yes to continue


  4. Select the "workload" components
    • Next you can select the workload "elements" to install
    • Here I've installed the most likely components I will need for creating Windows applications, also cloud applications If you forget one, you can run the installer again later to add other workloads
    • In my case, I also added Python and Node.js, because they support other projects I have.
    • Be sure and scroll down and add the database toolset as well
  5. Start the installer
    • By default the installer will "Install while Downloading. Install will start the process
    • It's time to get up and stretch. Read Chapters 1 from your textbook, maybe get a cup of coffee
    • I've seen this install process take anywhere between 30 minutes  or  2 hours. For these screenshots, I took a lightly used machine, uninstalled 3 different previous installations of Visual Studio to start a clean build, and it took close to 45 minutes for the installation

 

Once the product is installed, it will ask you to log in to Visual Studio. If you have a login that you already use on Microsoft Azure, you can use that. If you don't know what this is, then you can skip for now.
