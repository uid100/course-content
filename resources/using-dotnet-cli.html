Using the .NET CLI (Cross-Platform)

Getting started with C# programming using the .NET Command Line Interface (CLI). These instructions work on both Windows and macOS.

1. Check Installed SDKs

Open a terminal or command prompt and type the following command to list the installed .NET SDKs:

dotnet --list-sdks

2. Create a New Console App

In your terminal:

  • Create a new folder and navigate into it:
    • mkdir MyFirstApp
cd MyFirstApp
  • Create a new console project:
    • dotnet new console

Look at the files and folders that are added. This should be familiar.

3. Explore Available Templates

To see what other templates you can create with the CLI:

dotnet new list

4. Build and Run the Program

You can compile and run the program using:

dotnet build

or directly run it with:

dotnet run

5. Edit and Re-Run

Open the Program.cs file in a text editor and replace the contents with your own C# code. Save the file, then run:

dotnet run

6. Observe and Reflect

Notice that writing, compiling, and running your programs does not require an IDE. However, in most cases, using an IDE can make development easier and more efficient.
