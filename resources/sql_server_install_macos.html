SQL Server Installation Guide

created: 13-Jun-2024

Microsoft SQL Server supports Windows, Linux, and Azure platforms.

While there is not currently a reliable macos port, installation and use on a Docker container does provide an acceptable alternative.

The high-level steps are as follows:

  1. Download Docker and Azure Data Studio
  2. Create a Docker Account
  3. Download the SQL Server (container) image
  4. Run the container
  5. Connect to the database server
  6. Stop the container
