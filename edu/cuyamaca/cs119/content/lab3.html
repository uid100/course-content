Lab #3

This lab provides an opportunity to apply the concepts and principles presented in chapter 3 of the PLD text.

To save time, space and reading, all the nitty-gritty details of how to create a Python code file in IDLE, how to create the pseudocode file, etc. will be mostly left out. Please refer to the chapter 1 lab if you need a more detailed refresher on these steps. However, we will review problem analysis and pseudocode in quite a bit of detail in this exercise. We’ll focus more on the chapter concepts and the Python code needed to complete this problem as well as things you’ll need to know to complete the rest of this lab.

There are three exercises in this lab. Be sure to complete all three.

    Topics:
  • Creating a Python console application
  • Declaring variables
  • Assigning values to variables
  • Arithmetic operations
  • Selection structures and relational operators
  • Sequential code
  • Application testing; creating test data and verifying results
