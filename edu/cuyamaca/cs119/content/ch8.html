Chapter 8 More Object Concepts

In Chapter 7 you read that instances of classes are called objects which have attributes and methods. In this chapter we will discuss a very specific method common to all classes called the constructor.

A constructor is a special method that is called when an object is created. The constructor initializes the object's attributes and performs any other setup procedures required when an object is instantiated. We will also discuss the concept of overloading methods, which allows a class to have multiple methods with the same name but different parameters. This can be useful for providing different ways to perform the same operation depending on the context.

Additionally, we discuss the concept of inheritance and how specific groups of classes can be related to more generalized classes. Just as dogs are animals or cars and trucks and motorcycles and busses are all motor-vehicles.
