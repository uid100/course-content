3 Basic Programming Constructs

Programs are designed using common building blocks known as programming constructs. These programming constructs form the basis for all programs.

These are the three structures:

  • Sequence
  • Selection
  • Iteration
BBC Bitesize - Programming Constructs
Sequence

The sequence structure is the simplest of the three programming constructs. It is simply a series of instructions that are executed in the order in which they are written. This is the default mode of execution for most programming languages.

Example: 

print("Hello, World!")
name = input("What is your name? ")
print("Nice to meet you, " + name)
Selection

The selection structure, also known as a decision structure, allows the program to choose between two or more paths of execution based on a condition. This is typically implemented using if-else statements.

Example: 

age = int(input("Enter your age: "))
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")
Iteration

The iteration structure, also known as a loop, allows the program to repeat a block of instructions multiple times until a certain condition is met. Common types of loops include for loops and while loops.

Example: 

for i in range(5):
    print("This is iteration number", i)
