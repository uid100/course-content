Chapter notes

Programming in Python

We will be learning the Python programming language as part of this class.

We use Python because it's a fairly easy language for somebody who's new to programming, it allows us to apply most of the programming principles in the text, it's very marketable on a resume, and it's good preparation for other languages like Java, C++ and C#.

We will be practicing coding with Python, and you will learn a lot about making it work, but remember that this is a programming logic course. Understanding and applying the logic and patterns is going to be important.

The textbook is about programming in general, not specifically Python. Throughout the course you will be learning about important programming concepts and then applying them using examples in Python.

The linked videos in this module are provided by the textbook author and publisher. This a great way to hear an explanation of the concepts directly from the author!

The videos are not required, but they are a great resource to help you understand the concepts.

In addition to the videos, there is a link to the publishers summary powerpoint slides for this chapter. These are helpful for reviewing the concepts and for studying for the chapter quizzes.
