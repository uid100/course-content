Exercise 1

Find and fix syntax errors

Create a Python file named syntax_errors.py and copy the following code.

 

Each line that contains an error, add a hash (#) at the beginning of each line that contains an error to mark it as a comment and add another comment above it explaining what the error is, and then another line below that with the corrected code if possible.

For example:

        x = 5
        while x > 0
            print( x, end=' ' )
            x -= 1

would be replaced with:

        x = 5
        # while clause should be terminated with a color (':')
        # while x > 0
        while x > 0:
            print( x, end=' ' )
            x -= 1
