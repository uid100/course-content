Events and Event-handling

 

Event-driven programming

Event-driven programming is a programming paradigm where the flow of execution is determined by events such as user actions (mouse clicks, key presses), sensor outputs, or messages from other programs. In this module, we will explore how to implement event-driven programming concepts in Python, particularly in the context of graphical user interfaces (GUIs) and later, using web applications.
