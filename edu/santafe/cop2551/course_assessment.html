Learning Outcomes

From the course outline of record (COR), the program learning outcomes (PLOs) are defined as:

  • PLO1: Understand the fundamentals of programming with classes and objects.
  • PLO2: Write syntactically and semantically correct C# programs.
  • PLO3: Analyze program requirements and design solutions using OOP principles.

  • Lab 10: interfaces and abstract classes
  • Lab 11: Properties and fields
