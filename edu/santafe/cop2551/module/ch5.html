Introduction

In this chapter we will explore compound assignment operators and iteration statements in C#.

Compound assignment operators allow you to perform an operation and assign the result to a variable in a single step. For example, the += operator adds a value to a variable and assigns the result back to that variable, like this: 

x += 5; // equivalent to x = x + 5;

Iteration statements, also known as loops, allow you to execute a block of code multiple times based on a condition.

Compound Assignment Operators

Compound assignment operators in C# allow you to perform an operation on a variable and assign the result back to that variable in a single step. This can make your code more concise and easier to read.

These will likely be familiar to you, as they are used in most popular programming languages. The most common compound assignment operators in C# are:

  • += - Adds a value to a variable and assigns the result back to that variable.
  • -= - Subtracts a value from a variable and assigns the result back to that variable.
  • *= - Multiplies a variable by a value and assigns the result back to that variable.
  • /= - Divides a variable by a value and assigns the result back to that variable.
  • %= - Calculates the modulus of a variable with respect to a value and assigns the result back to that variable.

C# also supports compound assignment operators for bitwise operations. These operators allow the programmer to modify the data at the bit level, which can be useful for low-level programming tasks and logic. For now, we will stick with the arithmetic operators listed above.

If these operators are new to you, pay close attention to the syntax and how they work. Since they are all similar, once you understand one, the others will be easy to grasp.

If you don't fully grasp their meaning right away, don't worry, but stick with it. These are shorthand expressions that are in very common use in programming, and you will see them often.

As you become used to seeing them, you will find that they make your code cleaner and easier to read.

One thing to notice as you read through the chapter is that the variable's type dictates the rules of the operation. The same operator that is used to add integers together, also works for combining strings.
