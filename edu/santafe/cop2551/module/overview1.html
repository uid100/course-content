Like most modern programming languages, C# is an evolving platform. It was designed for and on the .NET platform. The .NET platform is a set of tools and libraries that allow you to create applications for a variety of platforms. The .NET platform is continually being updated and improved. The textbook is a great resource, but it is not the only resource. The Microsoft documentation is also a great resource. You can find the Microsoft documentation at https://learn.microsoft.com/en-us/dotnet/csharp/. The Microsoft documentation is a great resource for learning about the C# language and the .NET platform. It is also a great resource for learning about the Visual Studio IDE.

Because the .NET platform has its history on the Windows platform, many of the exampls in the textbook are Windows specific. The examples in the textbook are not always applicable to the Mac or Linux platforms. Most will work, but if you are running into difficulty, specific to one of the GUI application examples, read through it for information, but don't worry about always getting to work exactly as shown. The lab examples are tested to work on both Windows and macOS platforms, and are always relevant to the course content and objectives.
