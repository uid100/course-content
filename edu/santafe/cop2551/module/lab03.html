    Edit the source code

    Replace the source code with the following and update the header with your information.: 

    
/*
* Author:        First Last Name
* Date:          YYYY-MM-DD
* Class:         Santa Fe College, COP2551 OOP-1 with C#
* Assignment:    Lab 03 - Exercise #1 - Apples
* Description:   C# console application that ranks the results of
*                apple-harvesting using if-else statements.
*/

using System;

class Program
{
    Random random = new Random();

    // Declare variables
    int Alex = random.Next(1, 101);
    int Brian = random.Next(1, 101);
    int Colin = random.Next(1, 101);

    // Display the results
    Console.WriteLine($"Alex harvested {Alex} apples.");
    Console.WriteLine($"Brian harvested {Brian} apples.");
    Console.WriteLine($"Colin harvested {Colin} apples.");
    Console.WriteLine();

    // Determine the winner
    if (Alex > Brian && Alex > Colin)
    {
        Console.WriteLine("Alex is the winner!");
    }
    else if (Brian > Alex && Brian > Colin)
    {
        Console.WriteLine("Brian is the winner!");
    }
    else if (Colin > Alex && Colin > Brian)
    {
        Console.WriteLine("Colin is the winner!");
    }
    else
    {
        Console.WriteLine("It's a tie!");
    }

    Console.WriteLine("Press any key to exit.");
    Console.ReadKey();
}
  2. Explanation

    Understanding the code

    Random

    The program uses the Random class to generate random numbers between 1 and 100 for each of the three players. The program then compares the results using if-else statements to determine the winner.

    The code first creates a new instance of the Random class and assigns it to the variable random. We need to do this just once at the beginning of the program, so we can use it to generate random numbers.

    The Next() method is then called on the random object to generate a random number between 1 and 100.

    The Next() method takes two arguments: the minimum and maximum values for the random number. In this case, the minimum value is 1 and the maximum value is 101. The maximum value is exclusive, so the random number will be between 1 and 100.

    Boolean control statements

    This is a very simple program, but it demonstrates the use of the conditional if statement, the else if statement, and the else statement. The program also uses the Console.WriteLine()

    Remember that an else-if statement, just like an if statement requires a boolean expression. The else statement does not.

    Also, each of the boolean expressions evluates not just two integer values with a boolean operator, but also against a third integer value using the && operator. This is called a compound boolean expression.

    The program uses the familiar "Press any key..." statement to pause the program and wait for the user to press a key before exiting. This is a common practice in console applications to allow the user to see the output before the program closes.

  2. Design and Write the source code

    Moving across an X-Y grid

    Write a program that simulates moving across an X-Y grid. The program should start at the middle of the plot at position (0, 0) and keep track of the (X, Y) position. The tracker will move ten times, moving in one of four directions: up, down, left, or right. The program should display the new position after each move.

    Write a method called Move() and call it ten times. The method should take two parameters: the current X and Y position and return the new X and Y position as a tuple.

    The method should randomly select one of the four directions to move in, Up, Down, Left, or Right, and also randomly select the number of steps, between 1 and 10, to move in that direction. The method should then calculate and return the new X and Y position.

    Each time the method is called, display the direction and number of steps to move.

    Up, should increase the Y position by the number of steps, Down, should moved along the Y axis in the negative direction. Left and Right should move positive and negative along the X axis.

    Finally, calculate and display the total distance traveled. And the final distance from the starting position.

    The total distance traveled is the sum of all the steps taken.

    Use the Pythagorean theorem to calculate the distance from the starting position. The formula is distance = sqrt((x2 - x1)^2 + (y2 - y1)^2).

Exercise #3: Overloading methods

  1. Create the project

  2. Edit the source code

    Write overloaded methods to test the square of a number, using either an int or a double.

    Inside the Program class, write a new method called Square(). This method should take an int parameter and return an int value for the square of the number. The formula for the square of a number is square = number * number.

    Next, write another method called Square(). This method should take a double parameter and return a double value for the square of the number. The formula for the square of a number is square = number * number.

    The two methods should have the same name but different parameter types. This is called method overloading. The compiler will determine which method to call based on the type of the argument passed to the method.

    Edit the Main() method to call each of the methods you just created. Write lines (test cases) to test each of the methods and display the results.

    Run the program and test each of the methods. Test positive and negative numbers, and zero, to confirm that the methods work correctly.
