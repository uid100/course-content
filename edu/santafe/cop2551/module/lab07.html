Exercise #1: Traffic Light Controller (enum)

    • Solution: Lab07

    For this project (TrafficLight), use two classes, one for the LightController object and one for the TrafficLightApp. Each class should have its own source code file.

  2. Design and create the source code

    LightController.cs

    • In the LightController class file, define an enum called TrafficLight with the following values: Red, Yellow, and Green.
    • Create the LightController class with a parameterless constructor that initializes the current light to Red.
    • Add a method called ChangeLight() that changes the current light (held in an instance variable) to the next light in the sequence.
    • ChangeLight() uses a switch statement to determine the current light and change it to the next light in the sequence. If the light is Red, it changes to Green. If the light is Green, it changes to Yellow and has a 2 second delay before changing to Red. If the light is Yellow, it changes to Red. Define the delay value in milliseconds as a constant in the class.
    • To add a delay, add the following using directive at the top of the file: 
      
            using System.Threading;
      Then, use the Thread.Sleep() method to pause the program 
      
            Thread.Sleep(DELAY);
    • Add a method called GetCurrentLight() that returns the current light as a string.

    TrafficLightApp.cs

    • In the TrafficLightApp class file, create a Main() method that creates an instance of the LightController class.
    • In a continuous loop:
      • Display the current state of the traffic light
      • Prompt the user to select any key to continue, or 'Q' to quit
      • If the user selects 'Q' or 'q', exit the loop
      • Hint: Use a nullable value (ch.8) for the user input
      • Call the ChangeLight() method of the LightController object to change the light

    Do not forget to add the required header information for all of your source files!

    The program builds and runs without errors or warnings.

    The program instantiates the LightController object, exercises the instance methods, and displays the values to the screen.

  2. Design and Write the source code

    DayCounter.cs

    Create a class called DayCounter. Reminder, the classname should match the filename.

    Add an enum to the class file called DayOfWeek with the values: Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, and Saturday.

    In the class, create a Main() method that will be the entry point for the program.

    Get the current day of the week using the DateTime.Now.DayOfWeek property and store it in a variable called today.

    Prompt the user to enter a number of days to skip ahead (an integer). Use a nullable value to store the user input.

    Add a method called CalculateDay(int daysToSkip) that takes an integer parameter representing the number of days to skip ahead. The method should return the day of the week after skipping the specified number of days.

    Use the modulo operator to calculate the new day of the week. The formula is: (today + daysToSkip) % 7. This will give you the new day of the week as an integer, which you can then convert to the corresponding enum value.

    • How do you convert the result of the modulo operation back to the enum value?
    • Does CalculateDay() needs to be a static method?

Exercise #3: Estimate the value of pi using Monte Carlo method

  1. Create the project

    Add another console application project to the solution.

    • project name: PiEstimator
    • solution name: Lab07

    Name your project PiEstimator and add it to the solution.

    In statistical modeling, the Monte Carlo method is a computational algorithm that relies on repeated random sampling to obtain numerical results. The name is derived from the city of Monte Carlo in Monaco, which is famous for its casinos and gambling.

    In this exercise, you will use the Monte Carlo method to estimate the value of pi. The idea is to randomly generate points in a square that contains a quarter circle, and then calculate the ratio of points that fall inside the quarter circle to the total number of points generated. This ratio can be used to estimate the value of pi.

    Consider the following diagram:

    The quarter circle is inscribed in a square. The radius of the quarter circle is equal to the length of the sides of the square. The area of the quarter circle is equal to Ï€ * r^2 / 4, and the area of the square is equal to r^2. Therefore, the ratio of the area of the quarter circle to the area of the square is equal to Ï€ / 4. By randomly generating points in the square and counting how many fall inside the quarter circle, we can estimate the value of pi.

  2. Edit the source code

    PiEstimator.cs

    In this exercise, create a struct called Point that represents a point in a 2D space. The struct should have two properties: X and Y, both of type double. The struct should also have a constructor that takes two parameters for the X and Y coordinates.

    In the PiEstimator class, create a method called EstimatePi(int numberOfPoints) that takes an integer parameter representing the number of random points to generate. The method should return an estimate of pi as a double.

    Inside the EstimatePi() method, use a loop to generate the specified number of random points. For each point, generate a random X and Y coordinate between 0 and 1. Use the Random class to generate random numbers (double).

    Write another method to calculate the distance from the origin (0,0) to the point. Use the Pythagorean theorem to calculate the distance: distance = Math.Sqrt(point.X * point.X + point.Y * point.Y).

    Add a boolean method called IsInsideQuarterCircle(Point point). Pass each generated random point (struct) to this method to determine if the point is in the circle (true) or outside of it (false).

    From the loop, keep track of the number of points that fall inside the quarter circle. After the loop is complete, calculate the ratio of points inside the quarter circle to the total number of points generated. Use this ratio to estimate the value of pi using the formula: piEstimate = 4 * (pointsInside / totalPoints).

    Finally, return the estimated value of pi from the EstimatePi() method.

    In the Main() method, prompt the user to enter the number of points to generate for the estimation. Use a nullable value to store the user input. Call the EstimatePi() method with the user input and display the estimated value of pi to the user.

  3. Run and test the program

    Run the program and test it with different numbers of points to generate. The estimated value of pi should be close to the actual value of pi (3.14159...).
