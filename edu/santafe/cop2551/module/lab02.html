    Edit the source code

    Replace the source code with the following and update the header with your information.: 

    
/*
* Author:        First Last Name
* Date:          YYYY-MM-DD
* Class:         Santa Fe College, COP2551 OOP-1 with C#
* Assignment:    Lab 02 - Exercise #1 - Box Dimensions
* Description:   C# console application that handles 3D object dimensions and 
*                calculates the volume.
*/

using System;

class Program
{
    static double GetDimension(string name)
    {
        Console.Write($"Enter {name}: ");
        double value;
        while (!double.TryParse(Console.ReadLine(), out value) || value < 0)
        {
            Console.Write($"Invalid input. Please enter a positive number for {name}: ");
        }
        return value;
    }

    static double CalculateVolume(double height, double width, double depth)
    {
        return height * width * depth;
    }

    static void DisplayResults(double height, double width, double depth, double volume)
    {
        Console.WriteLine($"\nHeight: {height}");
        Console.WriteLine($"Width : {width}");
        Console.WriteLine($"Depth : {depth}");
        Console.WriteLine($"Volume: {volume}");
    }

    static void Main()
    {
        double height = GetDimension("height");
        double width = GetDimension("width");
        double depth = GetDimension("depth");

        double volume = CalculateVolume(height, width, depth);

        DisplayResults(height, width, depth, volume);
    }
}
  2. Explanation

    Understanding the code

    Most of the code and syntax should be familiar to you. The program uses the double data type to store the dimensions and volume of the box.

    Main()

    Main() is the entry point of the program.

    It calls the GetDimension() method three times to get the height, width, and depth of the box from the user.

    CalculateVolume calculates the volume of the box using the formula volume = height * width * depth.

    Finally, the DisplayResults() method is called to display the results to the console.

    Method signatures

    Notice the signature for each method.

    • Main()
    • GetDimension(string)
    • CalculateVolume(double, double, double)
    • DisplayResults(double, double, double, double)

    There should be one, and only one Main() method in your program. This is the entry point of the program. The program will not run without it.

    For each call to a methods, the compiler will look for a method with the same name and the same number and type of parameters. If it finds one, it will call that method.

    Return values and types

    From the method definitions, you can see that the GetDimension() method and the CalculateVolume() method return a double value. And in the code, there is a corresponding return statement in each of those methods.

    DisplayResults() and Main() are both declared with a void return type. This means that they do not return a value. The Display method is used to output to the console nothing is expected to be returned.

    Since Main() is the entry point of the program, it does not return a value to anything. This makes sense since there is no calling method to return to.

    GetDimension()

    While most of the code and syntax should be familiar to you, one line may look a little different and is worth explaining.

    while (!double.TryParse(Console.ReadLine(), out value) || value < 0)

    This line is a little more complex than the others. This line is part of a loop that keeps asking the user for input until they enter a valid positive number.

    step-by-step:

    1. Console.ReadLine()
      → Reads what the user types in the console as a string.
    2. double.TryParse(..., out value)
      → Tries to convert that string into a double number.
      → If it works, the result is stored in value.
      → It returns true if the conversion is successful, or false if it fails (like if the user types letters).
    3. while (!double.TryParse(...))
      → The ! means "not".
      → So loop will continue to be true when the input is not a number.
    4. value < 0
      → This checks if the number is negative.
    5. ||
      → The || operator means "or".
      → So the condition is true if either the input is not a number or it’s a negative number.

    So the full meaning is:

    "Keep asking the user until they type a number that is not negative."

  2. Write the source code (4 methods)

    Use parameters and return values from methods

    Inside the Program class, write a new method called AreaOfCircle(). This method should take a double parameter for the radius and an optional and return a double value for the area of the circle. The formula for the area of a circle is area = Math.PI * radius * radius.

    Next, write a method called CircumferenceOfCircle(). This method should take a double parameter for the radius and an optional parameter for the number of decimal places to round to. The method should return a double value for the circumference of the circle. The formula for the circumference of a circle is circumference = 2 * Math.PI * radius.

    Expression-bodied methods

    Next, write an Expression-bodied method called IsEven(). This method should take an int parameter and return a bool value indicating whether the number is even or not. Check the input parameter by dividing by 2 and checking the remainder. Return true or false.

    Remember that an expression-bodied method is a shorthand way of writing a method and does not require a return statement.

    Write another method called IsOdd() that does the same thing but checks if the number is odd.

    Optional parameters

    Finally, write a method called DisplayResults(). This method should take a string parameter for the name of the calculation, a double parameter for the result, and an optional int parameter for the number of decimal places to round to. The method should display the results in a formatted string. Use the Math.Round() method to round the result to the specified number of decimal places. If the optional parameter is not provided, use 2 decimal places by default.

    The method should display the results in a formatted string. For example: Area of circle with radius 5 is 78.54.

  3. Run and test the program

    Edit the Main() method to call each of the methods you just created. Write lines (test cases) to test each of the methods and display the results.

    Run the program and test each of the methods. Make sure to test the optional parameters by calling the methods with and without the optional parameters.

Exercise #3: Overloading methods

  1. Create the project

  2. Edit the source code

    Write overloaded methods to test the square of a number, using either an int or a double.

    Inside the Program class, write a new method called Square(). This method should take an int parameter and return an int value for the square of the number. The formula for the square of a number is square = number * number.

    Next, write another method called Square(). This method should take a double parameter and return a double value for the square of the number. The formula for the square of a number is square = number * number.

    The two methods should have the same name but different parameter types. This is called method overloading. The compiler will determine which method to call based on the type of the argument passed to the method.

    Edit the Main() method to call each of the methods you just created. Write lines (test cases) to test each of the methods and display the results.
