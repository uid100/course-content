Exercise #1: Box Dimensions

  1. Create the project

    Edit the source code

    Replace the source code with the following and update the header with your information.: 

    
/*
* Author:        First Last Name
* Date:          YYYY-MM-DD
* Class:         Santa Fe College, COP2551 OOP-1 with C#
* Assignment:    Lab 02 - Exercise #1 - Box Dimensions
* Description:   C# console application that handles 3D object dimensions and 
*                calculates the volume.
*/

using System;

class Program
{
    static void Main()
    {
        double height = GetDimension("height");
        double width = GetDimension("width");
        double depth = GetDimension("depth");

        double volume = CalculateVolume(height, width, depth);

        DisplayResults(height, width, depth, volume);
    }

    static double GetDimension(string name)
    {
        Console.Write($"Enter {name}: ");
        double value;
        while (!double.TryParse(Console.ReadLine(), out value) || value < 0)
        {
            Console.Write($"Invalid input. Please enter a positive number for {name}: ");
        }
        return value;
    }

    static double CalculateVolume(double height, double width, double depth)
    {
        return height * width * depth;
    }

    static void DisplayResults(double height, double width, double depth, double volume)
    {
        Console.WriteLine($"\nHeight: {height}");
        Console.WriteLine($"Width : {width}");
        Console.WriteLine($"Depth : {depth}");
        Console.WriteLine($"Volume: {volume}");
    }
}
  2. Explanation

    Understanding the code

    Most of the code and syntax should be familiar to you. The program uses the double data type to store the dimensions and volume of the box.

    Main() is the entry point of the program. It calls the GetDimension() method three times to get the height, width, and depth of the box from the user. CalculateVolume calculates the volume of the box using the formula volume = height * width * depth. DisplayResults() outputs the results to the console.

    Notice the signature for each method.

    • Main()
    • GetDimension(string)
    • CalculateVolume(double, double, double)
    • DisplayResults(double, double, double, double)

    There should be one, and only one Main() method in your program. This is the entry point of the program. The program will not run without it.

    For each call to a methods, the compiler will look for a method with the same name and the same number and type of parameters. If it finds one, it will call that method.

  2. Edit the source code

    Open the Program.cs file in the editor.

    Replace all the code with your solution.

    1 kilometer = 0.62 miles.

    This is the conversion factor that you are given. It must be in your code. Do not lookup or calculate another. Using other values will produce unexpected results.

    Note: The formula for converting kilometers to miles is:
    miles = kilometers * MILES_PER_KM

    Note: The constant MILES_PER_KM is a constant variable that holds the value of 0.62. It is a good practice to use constants for values that do not change in your program.

    It is also common practice to use all uppercase letters for constant identifiers (names).

    In the last exercise, you created integer variables to hold the values of the user inputs and the result. In this exercise, use double variables to store inputs, outputs and constants.

    To convert a string to a double, use the Convert.ToDouble() method.

    Give consideration to the names of your variables. Use descriptive names that indicate what the variable is used for. For example, km is a good name for a variable that holds the value of kilometers.

    Avoid using magic numbers (unnamed constant values) in your code. For example, instead of using 0.62 in your code, use a constant variable with a meaningful name, such as MILES_PER_KM. This makes your code easier to read and understand. It also makes it easier to change the value of the constant in the future if needed.
