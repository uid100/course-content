Exercise: 2-dimensional array grid

  1. Create the project

    • Solution: Lab08
    • Project: RotateGrid

  2. Edit the source code

    RotateGrid.cs

    • Within the Main() method, declare a 2D array of integers, the same number of rows as columns.
    • Write and call the method SquareGrid() to fill the array with sequential integers, from left-to-right and top-to-bottom starting from 1.
    • Write and call the method ShowMatrix() to display the contents of the array in a grid format.
    • Write and call the method FlipMatrix() to rotate the contents of the array, so that the first row becomes a mirror of the last row and the last row has the reversed data from the first row. See example below. 
    •         const int SIZE = 10; // size of the square grid
        matrix = SquareGrid(SIZE);
        ShowMatrix(matrix);
        FlipMatrix(matrix);
        ShowMatrix(matrix);

Exercise: Top-Ten Hits!

  1. Create the project

    Add another console application project to the solution.

    • project name: TopTen
    • solution name: Lab08

  2. Design and create the source code

    Top Ten!

    It's March, 1954, and the music charts are hot!

    1. Lollipop Serenade
    2. My Baby Wears Blue Suede Shoes
    3. Dance Floor Dreamboat
    4. Cruisin' Down Heartbreak Lane
    5. Twist My Heart Tonight
    6. He's Got the Jukebox Beat
    7. Moonlight at Makeout Point
    8. Shy Girl Shuffle
    9. Rockin' with Rosie
    10. We met at the Five and Dime

    Write a program to store, manipulate, and display the top-ten hit list.

    • The program will use an array of strings to store the list of songs in order.
    • Create an array and populate the list as shown above.
    • Write a method to print the numbered list, in order, to the screen (don't forget to number them).

    The next week, the top-ten list changes.

    • Updates!
      • A new song, Letter from a Lonely Teen moves to #4.
      • Hearts in Poodle Skirts is the new #7.
      • Cherry Cola Kisses debuts at #10.
    • Write a method to update the list with the new songs.
      • The new Insert() method will take three parameters: The array of songs, the index to insert at, and the song title.
      • The method will insert the song at the specified index, shifting the other songs down.
      • The method will return the updated array of songs.
      • Test the method by calling it with each of the song updates above.
      • Display the updated list of (10!) songs using the method you created above.

    • The Boy from Back Row High
    • Be-Bop Baby Blues
    • Three Steps to Love
    • Cryin'’' in the Sock Hop
    • Hearts in Poodle Skirts
    • You Stole My Saddle Shoes
    • Sweet Talk on Sunday Night
    • The Melody of Me and You.

    None of these songs in the test data are intended to be real.

  3. Run and test the program

    Run the program check the lists between the two weeks.
