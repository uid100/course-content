Exercise #3: Estimate the value of π using Monte Carlo method

  1. Create the project

    Add another console application project to the solution.

    • project name: PiEstimator
    • solution name: Lab07

    Name your project PiEstimator and add it to the solution.

    In statistical modeling, the Monte Carlo method is a computational algorithm that relies on repeated random sampling to obtain numerical results. The name is derived from the city of Monte Carlo in Monaco, which is famous for its casinos and gambling.

    In this exercise, you will use the Monte Carlo method to estimate the value of pi. The idea is to randomly generate points in a square that contains a quarter circle, and then calculate the ratio of points that fall inside the quarter circle to the total number of points generated. This ratio can be used to estimate the value of pi.

    Consider the following diagram:

    The quarter circle is inscribed in a square. The radius of the quarter circle is equal to the length of the sides of the square. The area of the quarter circle is equal to π * r^2 / 4, and the area of the square is equal to r^2. Therefore, the ratio of the area of the quarter circle to the area of the square is equal to π / 4. By randomly generating points in the square and counting how many fall inside the quarter circle, we can estimate the value of pi.

        (what will you call this one?)

        Refer back to the diagram above. If the distance from the origin to the Point is less than the radius, which is 1, the the point is inside the quarter circle.

  3. Run and test the program

    Run the program and test it with different numbers of points to generate. The estimated value of pi should be close to the actual value of pi (3.14159...).

    why does it work?

    Compare this diagram.

    As the number of points increases, the total points will fill the square. The area of the square is 1 * 1 = 1.

    The number of points inside the quarter circle will also increase. The area of the quarter circle is π * r^2 divided by 4, where r = 1.

    By extension, the ratio of the area inside the circle (number of points inside the circle) to the area of the square (total points) should be eventually be close to the value of π (3.14159...).
