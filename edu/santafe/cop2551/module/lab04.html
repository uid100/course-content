    Edit the source code

    First, take a look at a simple for loop. 

    
        using System;

        class Program
        {
            static void Main()
            {
                // For loop from 1 to 10
                for (int i = 1; i <= 10; i++)
                {
                    Console.WriteLine("Number: " + i);
                }
            }
        }

    Like the examples in the book, this simple program uses a for loop to iterate from 1 to 10, printing each number to the console.

    Another common use of for loops is in the context of nested loops.

    Replace the source code with the following and add the header information as you did in the previous labs.

    Build and run the program to see the output.

    The program uses a nested for loop to create a multiplication table from 1 to 5. The outer loop iterates through the numbers 1 to 5, and the inner loop multiplies each number by the current number in the outer loop.

    In this simple example, the limits are hard-coded. In a more complex program, these values should be set from variables to make the code more useful and flexible.

  2. Explanation

    Understanding the code

    The program uses a nested for loop to create a multiplication table. The outer loop iterates through the numbers 1 to 5, and the inner loop multiplies each number by the current number in the outer loop.

    In the for loop, remember that every loop has three parts:

    • Initialization: int i = 1 This sets the starting point of the loop.
    • Condition: i <= 5 This checks if the loop should continue.
    • Increment: i++ This increases the value of i by 1 after each iteration.

    It may be helpful to think of these as:

    1. Start
    2. Stop
    3. Step

Exercise #3: Guessing game using a while loop

  1. Create the project

    Add another console application project to the solution.

    • project name: Guess

    Lab04

        ├── Guess.csproj
        └── Program.cs

  2. Design and Write the source code

    Objective

    Create a simple number guessing game using a while loop.

    Instructions

    1. Assign a secret number between 1 and 100. Use an integer from a random number generator.
    2. Ask the user to guess the number.
    3. Keep looping until the user guesses correctly.
    4. For each incorrect guess, print:
      • "Too high!" if the guess is greater than the secret number.
      • "Too low!" if the guess is less than the secret number.
    5. When the user guesses correctly, print the result including the number of attempts it took to guess the number, and exit the loop.

    Just like using for loops, remember that every loop has three parts: START, STOP, STEP.

    These are not always as obvious as they are in a for loop, but they are still there. In a while loop, the STOP condition may be the most important part. The loop will continue until the STOP condition is met.

    Be sure that you can identify the START, STOP, and STEP parts of the loop in your code.

    When- not if- you end up with an infinite loop, you can stop the program by pressing Ctrl + C in the console window. It is usually because you forgot to change the value of the variable that is used in the STOP condition.

  3. Run and test the program

    Instead of hard-coding the upper limit of the random value, use a constant in your code. While you are testing, you can set this to a smaller value, like 10 or 20, to make it easier to guess the number and see the output.

    Here is an example of what the output might look like: 

    
        Guess the number between 1 and 10: 4
        Too low!
        Guess again: 9
        Too high!
        Guess again: 7
        Correct! You guessed it in 3 tries.

    The program should continue to prompt the user for guesses until they guess the correct number.

    If you get an infinite loop, you can stop the program by pressing Ctrl + C in the console window.

Review and Submit

  1. Review

    • for loop:
      • Used when you know exactly how many times to repeat.
      • Example: counting from 1 to 10.
    • nested for loops:
      • Used for structures with rows and columns.
      • Example: multiplication tables, printing patterns.
    • while loop:
      • Repeats as long as the condition is true.
      • Condition is checked before each iteration.
    • do-while loop:
      • Like while, but condition is checked after the first execution.
      • Guaranteed to run at least once.
