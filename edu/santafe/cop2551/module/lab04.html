    First, take a look at a simple for loop. 

    
        using System;
        using System;

        class Program
        {
            static void Main()
            {
                // For loop from 1 to 10
                for (int i = 1; i <= 10; i++)
                {
                    Console.WriteLine("Number: " + i);
                }
            }
        }

    Like the examples in the book, this simple program uses a for loop to iterate from 1 to 10, printing each number to the console.

    Another common use of for loops is in the context of nested loops.

    Replace the source code with the following and add the header information as you did in the previous labs.

    Build and run the program to see the output.

    The program uses a nested for loop to create a multiplication table from 1 to 5. The outer loop iterates through the numbers 1 to 5, and the inner loop multiplies each number by the current number in the outer loop.

    In this simple example, the limits are hard-coded. In a more complex program, these values should be set from variables to make the code more useful and flexible.

  2. Explanation

    Understanding the code

    The program uses a nested for loop to create a multiplication table. The outer loop iterates through the numbers 1 to 5, and the inner loop multiplies each number by the current number in the outer loop.

    In the for loop, remember that every loop has three parts:

    • Initialization: int i = 1 - This sets the starting point of the loop.
    • Condition: i <= 5 - This checks if the loop should continue.
    • Increment: i++ - This increases the value of i by 1 after each iteration.

    It may be helpful to think of these as:

    1. Start
    2. Stop
    3. Step

  2. Edit the source code

    Write a method called letterGrade() that takes a single integer parameter and returns a string. The method should return the letter grade for the given number. The letter grades are as follows:

    • A: 90-100
    • B: 80-89
    • C: 70-79

    The method should use a switch-case statement to determine the letter grade. The method should also use a default case to handle invalid input.
