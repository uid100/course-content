Before You Begin

File Extensions

Make sure your system is configured to show File Extensions

  • In Windows Explorer, go to the View tab and check the box for "File name extensions".
  • In Mac Finder, go to Preferences > Advanced and check the box for "Show all filename extensions".

Notice that the folders are numbered "lab01", "lab02", and so on. If you number these using only a single digit, then when we labs 10, 11, and 12, the natural ordering of your folders will be lab1, lab10, lab11, lab12, lab2, lab3, ...

This is because the computer sorts the folders in "alphabetical" order, not "numerical" order. So, lab1 comes before lab10, and lab2 comes after lab10.

To avoid this problem, use 2 digit numbers for the lab folders. This way, when we get to lab10, the folders will be sorted in the correct order: lab01, lab02, lab03, lab04, lab05, lab06, lab07, lab08, lab09, lab10, lab11.

Exercise #1: Multiply

  1. Create the project

    Follow the steps in chapter to for:
    Beginning programming with the Visual Studio 2022 environment
     to create a new project.

    Use the template drop-down filters for language: C# and type: Console. You can leave platform settings to All.

    Select the Console Application template and click Next.

    Set the folder (directory) to your new lab folder (Lab01), Name your project multiply. You can select to create the solution and project in the same directory or not. It is up to you. Later on, you can compare the differences. Click Next.

    Check the box for "Do not use top-level statements" and click Create.

    This will create a new folder called multiply with a new console application inside it.

    The folder will contain a Program.cs file and a .csproj file.

    The .csproj file is the project file that contains information about the project, such as the name, version, and dependencies.

    The Program.cs file is the main file that contains the code for the application.

     

    Run this code to see the output. You can start it by clicking the green arrow (Start button) in the toolbar or by pressing Ctrl + F5. Or you can use either of the Start options in the Debug menu.

    You have successfully created your first C# console application without Writing any code!

  2. Edit the source code

    Open the Program.cs file in the editor. You will see a lot of code that you did not write. This is the code that was generated for you by Visual Studio.

    Notice there are three "blocks" of code in the file.

    The first block is the namespace. It is named the same as the project.

    Inside this, the second block is the class Program. This is the main class of the program. It contains the entry point of the program and it is contained in the namespace.

    The third block is the method Main(). This is the main method of the program. It is where the program starts executing.

    When you run the program, it will execute the code in the Main() method and print the output, currently "Hello World!" to the console.

    Use the editor to replace all the code with the following: 

                                        using System;

                                    class Program
                                    {
                                        static void Main()
                                        {
                                            int number1;
                                            int number2;

                                            userInput = Console.ReadLine();
                                            number2 = Convert.ToInt32(userInput);
                                    
                                            product = number1 * number2;
                                    
                                            Console.WriteLine($"The product of {number1} and {number2} is {product}.");
                                        }
                                    }
