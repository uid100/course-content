Java Data Structures and Generic Collections

In this activity you are going to prove that you can write programs that use LinkedList, ArrayList, Queue, and Stack, and that you can write code to create a binary search tree.

Setting Up Eclipse

Be sure to have the latest version of the project from the repository before starting.

Get the starter code from: https://github.com/uid100/JavaDataStructures

  • Download the code and open it in Eclipse. There are more detailed instructions in the README.md file in the GitHub repository.
  • The test cases are provided in the src folder. You will need to uncomment each test case and provide the code for the logic to make the test pass.

Hint:

To uncomment blocks of code all at once in Eclipse, select the commented code and use shift+'/' or command+'/' on a Mac. This will uncomment the code. Use the same key combination to comment the code.

Don't forget to go back and refactor (clean up) your code after you get it working. This is a good habit to have.
  • When you are finished, zip the project and submit to assignment page as you did in the previous module.

As before, you will be given the test cases and you will need to write the code and add the classes to make the tests pass.

  1. Open and run the tester. You will see that the code does not compile due to errors.
  2. Open LinkedListProducer.java and LinkedListConsumer.java and complete the parts of the code marked TODO to work correctly.
  3. In the main menu, select Run -> Run If you correctly edited the producer and consumer classes, the test will pass.
  4. Work through each of the test cases and continue to implement the needed method.
    Starter code for the producer and consumer classes for LinkedList, ArrayList, Queue, and Stack are provided to you.

The challenge is complete when you have all tests succeeding.

The README.md file in the GitHub repository has more detailed instructions.
