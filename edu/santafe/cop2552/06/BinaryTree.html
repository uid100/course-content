Overview

A tree is a non-linear data structure well-suited for searching and sorting. Binary trees specifically have at most two child nodes for each parent.

You can try a binary search tree here: Binary Search Tree Visualization (https://www.cs.usfca.edu/~galles/visualization/BST.html). For instance, try inserting names of people you know.

Use Cases

Trees are helpful if you want to be able to sort and search data efficiently. You can use the Java classes TreeMap and TreeSet to implement trees. Note: Do not use these classes in the programming challenge for this module; instead, write your own classes to implement the necessary tree!
