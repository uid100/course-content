Working with Java Arrays

In this activity you are going to demonstrate that you can write programs that use arrays.

You are given a set of programmed tests that each verify a static method. Your challenge is to implement the methods so that the methods pass; you will need to read the testing code to determine what methods you need to write, and what each of the methods should do.

  1. Get the latest build from GitHub
    https://github.com/uid100/Java2DimensionalArrays
  2. Import CISC191M2ArrayChallenges into Eclipse.
  3. Import the file into Eclipse
  4. Rename the project to be CISC191M2ArrayChallenges_YourName
  5. Open CISC191M2ArrayChallenges -> src -> TestArrayChallenge.java
  6. In the main menu, select Run -> Run
  7. Then you will see that the first test ran but did not pass.
  8. Then open CISC191M2ArrayChallenges -> src -> ArrayChallenge and change the contains() method to work correctly.
  9. When you think you have it, go back to TestArrayChallenge.java and run the tests again.
  10. When you have the contains method working correctly (the test passes), uncomment the next test method and continue to implement the needed method.

The challenge is complete when you have all tests succeeding.

If you get stuck on a particular method, comment it out, and move on to the next. You can come back it later, perhaps after getting some help.

When you have completed the assignment, export the project from Eclipse to a zip file and submit it to the assignment page as you did in the previous module.
