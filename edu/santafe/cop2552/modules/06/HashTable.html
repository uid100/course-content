Overview

A hash table is a 2-D data structure that associates a set of keys with values. You can think of it like an array, but instead of using an index to access an element, you use a key to access an element. Keys can be any type; they are not restricted to numbers.

Hash Table Example
Key Value
Maria 858-555-1234
Jose 619-555-8472
愛子 619-555-3829
Raul 858-555-8391

You can try a hash table here: Open Hashing (https://www.cs.usfca.edu/~galles/visualization/OpenHash.html). Try inserting two numbers that are bigger than 12, for instance 13 and 15 (can you see the pattern?).

Use Cases

Hash tables can be used for "translations" between words/objects. In Java you can use the Hashtable class to implement a hash table.
