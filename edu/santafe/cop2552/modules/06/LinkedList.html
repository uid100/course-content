Overview

A LinkedList is an ordered collection of data.

Image from https://www.geeksforgeeks.org/implementing-a-linked-list-in-java-using-class/

Use Cases

Linked lists are useful when you have ordered data; typically data is removed or added at the beginning or end of the collection, but it is possible to insert in the middle. This is efficient for collections that grow/shrink in length, but inefficient at accessing random values. A Linked List can be used to implement a stack or a queue.
