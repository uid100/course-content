Overview

A stack is an example of a Last-In-First-Out (LIFO) system. Imagine a stack of pancakes; the top pancake was the Last-In, but it will be the First-Out when someone takes another pancake.

Image from http://www.dailycodebuffer.com/java-stack-class-tutorial/

You can try a stack here: Stack (Array Implementation) (https://www.cs.usfca.edu/~galles/visualization/StackArray.html)

Use Cases

Java method calls use a stack. Java has a Stack class that can be used for implementing the stack data structure.
