Overview

An ArrayList is like an array whose size can grow or shrink as needed, unlike an array whose size is fixed.

Use Cases

ArrayLists are useful when you don't know how many items will need to be stored in an array, and can be used anywhere that an array might be used. Like arrays, ArrayLists can only hold one type of data.

Java Arraylist Class

To use an ArrayList:

import java.util.ArrayList;

To declare and instantiate an ArrayList:

ArrayLists can hold any type; just specify the type when declaring and instantiating the ArrayList

To create an ArrayList of type String:

ArrayList<String> myArrayList = new ArrayList<String>();

To create an ArrayList of type MyClass

ArrayList<MyClass> myArrayList = new ArrayList<MyClass>();

ArrayLists have methods to add, get, set, and remove items. To determine how many items are in an ArrayList, use the size method.

Java Code Example

import java.util.ArrayList;
import java.util.Collections;

public class ArrayListExample
{
   public static void main(String[] args)
   {
      // create an ArrayList to hold Strings
      ArrayList<String> computerScientists = new ArrayList<String>();
      
      // add items to the ArrayList
      computerScientists.add("Grace Hopper");
      computerScientists.add("Bill Gates");
      computerScientists.add("Frances Allen");
      computerScientists.add("Ada Lovelace");
      computerScientists.add("Valerie Taylor");
      
      // sort the list
      // If using a custom type, it needs to implement Comparable
      Collections.sort(computerScientists);
      
      // print the list
      System.out.println("Alphabetized list:");
      for (int i = 0; i < computerScientists.size(); i++)
      {
         // use "get" to retrieve the item at location i
         System.out.println(computerScientists.get(i));
      }
      
      // remove the middle item (0-indexing)
      computerScientists.remove(2);
      
      // change the first item
      computerScientists.set(0, "Your Professor");
      
      // print the list using a for-each loop
      System.out.println("\nMiddle item removed, first item changed:");
      for(String name: computerScientists)
      {
         System.out.println(name);
      }
   }
}
