Upon completion of the course, the student is expected to demonstrate practical applications of software engineering methodologies and practices as outlined below:

  • Develop and implement a project assessment and needs analysis
  • Develop a completed project plan
  • Create and implement a project architecture document
  • Develop a software project based on the needs analysis, project plan and the project architecture documents
  • Test and implement the software project
  • Develop appropriate user documentation
  • Develop an update plan
