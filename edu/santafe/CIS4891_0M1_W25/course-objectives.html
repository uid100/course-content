Upon completion of the course, the student is expected to demonstrate practical applications of software engineering methodologies and practices as outlined below:

  • Master the concepts and principles of cloud computing.
  • Explain the basic concepts of cloud computing.
  • Identify and distinguish cloud types.
  • Explain methods of managing cloud computing systems.
  • Identify standards, best practices, and policies of cloud computing management.
  • Describe cloud services; SaaS, PaaS, IaaS, IDaaS.
  • Perform a business impact analysis.
  • Identify advantages and disadvantages of cloud-based storage.
  • Explain best practices when collaborating in the clouds.
  • Demonstrate professional level skills in the areas of disaster recovery and business continuity and the cloud.
  • Explain service-oriented architectures and web services.
  • Master techniques of using Infrastructure-as-a-Service, (IaaS), Platform-as-a-Service (Paas) and Big Data systems.
  • Master the techniques of developing, deploying, and managing cloud applications major topics: Introduction background and history of cloud computing, cloud computing concepts, cloud computing models with virtualization background and history of virtualization, virtual machines, virtual networks, virtual storage IaaS system architecture, IaaS programming, PaaS System Architecture, PaaS Programming, Big Data system architecture and Big Data programming.
