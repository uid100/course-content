REQUISITES ANALYSIS
Knowledge of basic computer organization, problem solving techniques, and data processing systems
  1.     Course: CISC 181      Identify and describe the components of a computer.
  2.     Course: CISC 181      Differentiate between the most widely used business software application packages and explain how they are used.
  3.     Course: CISC 181      Solve problems with a spreadsheet using relative and absolute addressing where appropriate to enhance copying formulas.
  4.     Course: CISC 181      Use date functions, vlookup functions, statistical functions, if functions, nested functions, and range names to create spreadsheets.
  5.     Course: CISC 181      Demonstrate solutions using spreadsheets with pivot tables, pivot charts, subtotals, and grouping.
  6.     Course: CISC 181      Analyze spreadsheet data using data base functions, advanced filter, data tables, goal seek, and solver.
  7.     Course: CISC 181      Contrast the various types of storage devices used on computers.
  8.     Course: CISC 181      Compare and contrast the various operating systems used on modern computers and demonstrate the use of modern communication networks.
  9.     Course: CISC 181      Examine the emerging concepts of the Internet and the World Wide Web.
  10.     Course: CISC 181      Use a database to design tables, establish relationships, create queries, and calculated fields.
  11.     Course: CISC 181      Examine how information systems are designed and how the systems life development cycle applies to business.
  12.     Course: CISC 181      Solve problems using a programming language to include interface design, variables, constants, calculations, and decisions.
  13.     Course: CISC 181      Analyze the importance of security in computer systems and how information privacy concerns and business ethics are incorporated in a computer business system.
  14.     Course: CISC 181      Define multimedia and examine multimedia's place in computer business systems applications.
  15.     Course: CISC 181      Apply the principles of effective collaboration through a process of feedback and iteration.
