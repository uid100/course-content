8-week course format
This course schedule has been condensed to fit the college's accelerated 8-week term format
There is a lot of content to cover in a short amount of time, which will require a significant time commitment.
Students should expect to spend 10-15 hours per week on this course.
The course outline will loosely follow the sequence of topics in the texztbook.
Topics
Week 1
module 1. Appications and Sequential Data Processing
module 2. Control Structures: Conditional Execution
Week 2
module 3. Control Structures: Iteration (For- and While- loops)
module 4. Functions and Modular Code
Week 3
module 5. Testing and Debugging
Week 4
module 6. Collections: Arrays, Lists and Tuples
Week 5
module 7. File IO and Exception Handling
Week 6
module 8. Dictionaries and Sets
Week 7
module 9. Classes and Objects
module 10. Inheritance
Week 8
module 11. APIs and Web Applications
Pattern and due dates
The assignments with each module will typically be due on Wednesday or Saturday night at 1159P. Some assignments will have 1/2 week deadlines, and some will have a full week.
The details of all the due dates can be found at the bottom of this syllabus page, as well as on the course calendar within Canvas.
