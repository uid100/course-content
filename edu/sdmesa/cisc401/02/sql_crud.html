Introduction to SQL

SQL (Structured Query Language) is a standard language for managing and manipulating databases.

A foundational concept in data science is known as CRUD operations.

  • Create
  • Read (or Retrieve)
  • Update
  • Delete

Starting with the basics of SQL operations, we will look at how to create tables and add and retrieve data.

Creating a Simple Table

To create a table in SQL, we use the CREATE TABLE statement. Here is an example of creating a simple table named students:

CREATE TABLE students (
    id INT PRIMARY KEY,
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    age INT
);

CRUD Operations

CRUD operations refer to the four basic operations we can perform on a database: Create, Read, Update, and Delete.

Create

To insert data into a table, we use the INSERT INTO statement. Here is an example:

INSERT INTO students (id, first_name, last_name, age)
VALUES (1, 'John', 'Doe', 20);

Read

To retrieve data from a table, we use the SELECT statement. Here are a few examples:

-- Select all columns from the table
SELECT * FROM students;

-- Select specific columns from the table
SELECT first_name, last_name FROM students;

-- Select with a condition
SELECT * FROM students WHERE age > 18;

Update

To update existing data in a table, we use the UPDATE statement. Here is an example:

UPDATE students
SET age = 21
WHERE id = 1;

Delete

To delete data from a table, we use the DELETE FROM statement. Here is an example:

DELETE FROM students
WHERE id = 1;

Conclusion

In this lecture, we have covered the basics of SQL operations, including creating a simple table and performing CRUD operations. These fundamental skills are essential for managing and manipulating databases effectively.
