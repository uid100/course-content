CISC 401 Database Management and Design for Healthcare professionals

This course provides healthcare professionals with an introduction to database programming concepts and skills essential for managing and utilizing healthcare data. It covers database design, SQL programming, data integration, and practical applications in the healthcare industry.

Course Objectives:

  • Understand the fundamentals of databases and database management systems (DBMS)
  • Learn to design and implement relational databases
  • Gain proficiency in SQL for data manipulation and querying
  • Explore data integration techniques and healthcare-specific standards
  • Apply database programming skills to real-world healthcare scenarios

Introduction to Databases

  • Overview of databases and DBMS
  • Importance of databases in healthcare
  • Types of databases (relational, NoSQL, etc.)
  • Introduction to database programming
