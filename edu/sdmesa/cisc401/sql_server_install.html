SQL Server

installation guide

(13-Jun-2024)

Note: A new Windows PC build image (vm) was used for this walk-through. No pre-existing software is installed or assumed on this (Windows 10 pro) system at the beginning of the install. These instructions and steps are subject to change, but are current as of this date.

  1. Download installer

    Navigate to the Microsoft download link for SQL Server.

    https://www.microsoft.com/en-us/sql-server/sql-server-downloads

    Scroll down the page and find the link to download the Developer version of the product.

    Developer
    SQL Server 2022 Developer is a full-featured free edition,
    licensed for use as a development and test database in a
    non-production environment.

  2. Run the installer

    1. Launch the downloaded installation utility,
    2. Select the Basic installation,
    3. Read and accept the license terms for the developer edition of SQL Server
    4. Confirm the install location. In most cases, the default is recommended
    5. Click the [Install] button to begin

    For reference, in this case, using a modest system configuration and relatively fast internet access, the automated installation process, including acquiring the setup files, installing and configuring the software completed in under 10 minutes.

  3. Install SQL Server Management Studio

    At the installation complete screen, select the option to install SQL Server Management Studio [Install SSMS].

    This will open a link in your browser to download the setup utility for SSMS. After downloading the setup utility file, close the browser and launch the installer from your downloads folder

    Accept the default installation location. The install will typically take another 5 minutes or so.

  4. (Optional) Enable the network service.

    At this point SQL Server is installed and can be accessed from the local (Windows) machine.

    In order to access the database from another computer on your network, enable the service and allow access through the firewall.

    1. Configure SQL Server
      From the Windows Start Menu, find the newly installed:
      Microsoft SQL Server 2022 and
      SQL Server 2022 Configuration from the drop-down menu.

    2. Select SQL Server Network Configuration > Protocols for MSSQLSERVER, right-click TCP/IP and Enable.

    3. Restart the service.

  5. (Optional) Configure the network firewall.

    The SQL Server instance will support (TCP) network access on port 1433. By default, the Windows firewall will block access to any network ports that are not explicitly opened. The final step to enable network access to your database is to configure the firewall to allow access to that port.

    This can be accomplished quickly and simply from the command line, powershell, or management console. 

                        netsh advfirewall firewall add rule name = SQLPort dir = in protocol = tcp action = allow localport = 1433 remoteip = localsubnet profile = DOMAIN

    Refer to these instructions for more information: learn.microsoft.com
