Building Database Applications

Exercise: Create a Database Application using SQL Server and Python

Objective

In this exercise, you will create a simple database application using SQL Server as the database management system (DBMS) and Python as the programming language. This application will demonstrate the basic functionalities of connecting to a database, executing queries, and displaying results using Python.

Create Database and Populate Tables

1. Create a new database named: scheduledb using Azure Data Studio

2. Download, inspect and run the attached SQL scripts to create the database schema and populate the tables with sample data.

  • CreateTables.sql
  • InsertData.sql

3. Add another record to the Staff table with your first and last name.

Query the Staff table to verify that the new record has been added successfully and to view the assigned staff_id.

Record this query and response in a new MS Word document.

Step 2. Download and install Python and the pyodbc module

1. Download and install Python from the official website: Python Downloads

2. Open a command prompt and install the pyodbc module using the following command: 

            pip install pyodbc

3. Download the provided Python script dbapp.py and save it to your local machine.

  • dbapp.py

Step 3. Run the Python script to demonstrate database connection

1. Find and run the app: IDLE which installed with Pyhton.

On MacOS, it is likely located in a folder named Python3.x in the Applications folder.

On Windows, use the Start menu to find and run newly installed IDLE.

2. Open the dbapp.py script in IDLE and review the contents.

3. Run the script by selecting Run > Run Module from the menu or pressing F5.

Enter First and Last names in the app when it launches to see the work schedule for that staff member.

Display work schedule for 3 different staff members. Add this to your MS Word document.

Login with your (added) name and staff id. What displays for your work schedule?

Submit the Word Document demonstrating the working database application.
