requisites (CISC 181)
description
Identify and describe the components of a computer
Differentiate between the most widely used business software application packages and explain how they are used
Solve problems with a spreadsheet using relative and absolute addressing where appropriate to enhance copying formulas
Use date functions, vlookup functions, statistical functions, if functions, nested functions and range names to create spreadsheets
Demonstrate solutions using spreadsheets with pivot tables, pivot charts, subtotals and grouping
Analyze spreadsheet data using data base functions, advanced filter, data tables, goal seek and solver
Contrast the various types of storage devices used on computers
Compare and contrast the various operating systems used on modern computers and demonstrate the use of modern communication networks
Examine the emerging concepts of the Internet and the World Wide Web
Use a database to design tables, establish relationships, and create queries and calculated fields
Examine how information systems are designed and how the systems life development cycle applies to business
Solve problems using a programming language to include interface design, variables, constants, calculations, decisions and conditions
Analyze the importance of security in computer systems and how information privacy concerns and business ethics are incorporated in a computer business system
Define multimedia and examine multimedia's place in computer business systems applications.
