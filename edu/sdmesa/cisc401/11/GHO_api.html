Global Health Observatory API

Introduction

The Global Health Observatory (GHO) is the World Health Organization's portal providing access to data and analyses for monitoring the global health situation.

The GHO data repository contains an extensive list of indicators, which can be selected by theme or through a multi-dimension query functionality.

It is the World Health Organization's main health statistics repository.

The GHO provides access to data from WHO's Member States. The data repository is updated on a regular basis to provide users with the most recent data available.

The GHO data repository provides access to over 50 datasets on priority health topics including:

  • mortality and burden of diseases
  • the Millennium Development Goals:
    • nutrition
    • child health
    • maternal and reproductive health
    • immunization
    • HIV/AIDS
    • tuberculosis
    • malaria
    • neglected diseases
    • water and sanitation
  • non communicable diseases and risk factors
  • epidemic-prone diseases
  • health systems
  • environmental health
  • violence and injuries
  • equity
  • others

The GHO data repository is available through a number of flexible tools, including an API that allows users to access the data in a variety of formats, including XML, JSON, and CSV.

The GHO API provides access to a wide range of data, including data on health workforce, health systems, and health financing. The API allows users to filter the data by country, date, and other dimensions, and to retrieve the data in a variety of formats, including XML, JSON, and CSV. The API is designed to be flexible and easy to use, and provides access to a wealth of data on global health.

Life Expectancy Analysis

Query: Retrieve "Life expectancy at birth" (e.g., WHOSIS_000001).

Analysis: What are the trends in life expectancy across different regions over the last decade? Are there significant differences by gender?

https://ghoapi.azureedge.net/api/WHOSIS_000001?filter=data(TimeDimensionBegin) ge 2013-01-01
